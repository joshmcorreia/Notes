# Bot Detection Tests
Below is a list of websites that can be used to test if you're seen as a bot.

- [CreepJS](https://abrahamjuliot.github.io/creepjs/)
- [Sannysoft](https://bot.sannysoft.com/)
- [Puppeteer Bot Check](https://github.com/prescience-data/puppeteer-botcheck)
- [Pixelscan](https://pixelscan.net/)
- [Fingerprint](https://fingerprint.com/demo/)
- [FingerprintJS](https://fingerprintjs.github.io/fingerprintjs/)
- [Cover Your Tracks](https://coveryourtracks.eff.org/)
- [Am I Unique?](https://amiunique.org/fingerprint)
- [BrowserLeaks](https://browserleaks.com/canvas)
- [DeviceInfo](https://www.deviceinfo.me/)
