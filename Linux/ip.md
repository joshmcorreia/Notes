# ip

---

## List all IP addresses
```
$ ip a
```

---

## List IPv4 addresses
```
$ ip -4 a
```

---

## List IPv6 addresses
```
$ ip -6 a
```

---

## List routing table entries
```
$ ip route
default via 192.168.1.1 dev enp3s0
192.168.1.0/24 dev enp3s0 proto kernel scope link src 192.168.1.2
```
