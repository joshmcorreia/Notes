# String interpolation:

### Python3:
Note that these are specifically called "f-strings".
``` python
my_name = "Josh"
message = f"My name is {my_name}."
```

### JavaScript:
``` js
var my_name = "Josh"
var message = `My name is ${my_name}.`
```

### C#:
``` csharp
string my_name = "Josh";
string message = $"My name is {my_name}.";
```

### bash:
``` bash
my_name="Josh"
echo "My name is $my_name"
```
